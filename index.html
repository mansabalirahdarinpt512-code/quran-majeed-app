While surahFatihaWords[endIdx] matches transcript: endIdx++ 3. WORD STATE MACHINE States: idle | active | blinking | blue | correct | locked idle → active when previous word becomes correct (batchAdvance) active → correct when batchAdvance matches (from interim/final result) active → blinking when silence timer fires (4000ms) = wrong detected blinking → blue after Qari audio plays (afterAudio callback) blue → active after resumeAfterSpeaking (processingRef=false) any(not correct) → locked when wrong fires (lockFuture from wrong word forward) locked → idle when batchAdvance marks a correct word (unlock all after) Visual per state: idle: color: word.color, opacity: 0.78 active: color: word.color, textShadow: 0 0 22px {color}CC, transform: scale(1.12) correct: color: #1A7A4A, textShadow: green glow blue: color: #1565C0, textShadow: blue glow blinking: color: #C0392B, CSS class blink-red (infinite red flash animation) locked: color: rgba(140,140,140,0.18) — ghost text (use rgba NOT opacity so child 🔒 renders at full opacity) + absolute-positioned 🔒 emoji (fontSize:9, bottom:-4, centered) WRONG-WORD FEEDBACK FLOW Triggered ONLY by 4-second silence (onSilence callback). Never by speech events. handleWrong(word): 1. processingRef.current = true 2. lockFuture(word.id) ← all words after this get "locked" + 🔒 3. setOne(word.id, "blinking") ← red flash starts 4. pauseForSpeaking() ← mic ignores input (does NOT abort mic) 5. if wrongSpokenRef.has(word.id): ← Already spoke once. Blink 1.2s then: setOne(word.id, "active"), processingRef=false, resumeAfterSpeaking() return 6. wrongSpokenRef.add(word.id) 7. playWordAudio(getWordAudioUrl(word), afterAudio, afterAudio) ← If CDN fails, afterAudio is called directly (NO TTS fallback) afterAudio(): setOne(word.id, "blue") setStatusMsg(t.tryAgain) processingRef.current = false resumeAfterSpeaking() First correct after wrong: batchAdvance calls wrongSpokenRef.current.delete(word.id) → flag cleared instantly. If wrong fires again on the next attempt, Qari speaks again (fresh attempt = fresh chance).
